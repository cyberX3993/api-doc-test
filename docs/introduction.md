# Introduction

Welcome to the Student Management API documentation. This API enables schools and educational organizations to manage student records, courses, attendance, and grades through a secure REST interface.

## Base URL

```text
https://api.school.example.com/v1
```

## API Principles

- RESTful resources and standard HTTP verbs
- JSON request and response bodies
- HTTPS-only transport
- JWT Bearer authentication
- ISO8601 timestamps
- UUID resource identifiers
- camelCase JSON properties

## Resources

- `auth` — login and token management
- `students` — student profiles and lifecycle management
- `courses` — available academic programs
- `attendance` — daily attendance records

For detailed endpoint definitions, see [API Reference](./api-reference.md).
