# Security Policy

## Report a Vulnerability

If you discover a security issue in the Student Management API documentation or spec, please open an issue or contact the maintainers immediately.

## Supported Versions

This repository contains the initial `v1` documentation for the API.

## Authentication

The API uses JWT Bearer tokens for authentication. Always transmit tokens over HTTPS.

## Best Practices

* Do not expose access tokens in public logs.
* Rotate refresh tokens after they are used.
* Enforce strong password policies on login endpoints.
* Rate-limit requests to protect against abuse.
